# Time Off Request Web App

Google Apps Script로 만든 휴가 신청 웹 애플리케이션입니다.

## 주요 기능

- 📝 간편한 휴가 신청 폼
- 📧 자동 이메일 알림
- 📊 휴가 현황 관리
- ✅ 승인/반려 처리

## 사용 방법

1. Google Workspace Marketplace에서 설치
2. 휴가 신청 폼 열기
3. 필요한 정보 입력 후 제출

## 기술 스택

- Google Apps Script
- Google Sheets (데이터 저장)
- HTML/CSS/JavaScript
